Recommended diagnostic tools
	•	Duplicate Finder
	•	exact duplicate names
	•	normalized-name near matches
	•	same domain used by multiple master orgs
	•	likely duplicate aliases
	•	Structure Coverage Report
	•	facilities with no structure scan ever run
	•	facilities scanned but still unresolved
	•	facilities with no parent
	•	orgs with no ultimate parent
	•	Relationship Integrity Check
	•	orphaned child records
	•	circular relationships
	•	parent-child type mismatches
	•	orgs marked enterprise that still have parents
	•	Confidence Review Queue
	•	low-confidence structure matches
	•	LLM-assisted matches awaiting review
	•	manually entered master orgs with weak evidence
	•	promoted records missing verification
	•	Domain Diagnostics
	•	missing website/domain
	•	duplicate root domains across different orgs
	•	suspicious mismatches between org name and domain
	•	invalid or malformed domains
	•	Alias / Naming Diagnostics
	•	orgs with no aliases
	•	aliases that match another master org’s canonical name
	•	naming collisions across hospitals, campuses, and systems
	•	overlong or messy names needing normalization cleanup
	•	Seed Data Health
	•	seeded orgs with no relationships
	•	seeded systems missing child facilities
	•	seeded facilities not linked to enterprise
	•	seeded records never reviewed since import
	•	Freshness / Revalidation Queue
	•	orgs not reviewed in 90/180/365 days
	•	facilities whose structure was never revalidated
	•	records created from workspace approvals but never audited
	•	stale external-source evidence
	•	Source-of-Truth Diagnostics
	•	master records created manually
	•	master records promoted from workspace

	•	records where sources disagree
